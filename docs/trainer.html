trainer module¶

trainer.train_model(filename, num_epoch, step_epochs, batch_size, lr, save_name, engineered_features=False, multiple_dataset=False)¶

Train a model on the given dataset

Written by Leo Nguyen. Contact Xenovortex, if problems arises.

Args:

filename (string): name of h5 file containing dataset num_epoch (int): number of epochs step_epochs (list): list of epoch number, where learning rate will be reduce by a factor of 10 batch_size (int): batch size lr (float): learning rate save_name (string): name under which to save trained model and results engineered_features (bool, optional): contenate engineered features to vectorized sentence multiple_dataset (bool, optional): use multiple datasets
